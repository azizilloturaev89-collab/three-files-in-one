# three-files-in-one
This program takes three files and writes their elements into a fourth file in order: first element of the first file, first element of the second file, first element of the third file, and so on.
# Merge Three Binary Files in C++

This program creates three binary files with random numbers and combines their elements into a fourth binary file.

## How it works
- Generates random numbers
- Writes them into three binary files
- Reads elements from the files
- Writes them into a fourth file
- Displays the final result

## Language
C++
